# PostgreSQL Concurrent Booking System

A PostgreSQL-native booking system built to study concurrency control, transactions, and database performance under high concurrent load.

## Project Goal

Build a booking system that guarantees:
- Zero overselling
- Zero double charges
- Safe concurrent booking using PostgreSQL

## Features (Planned)

- Seat booking system
- Seat hold mechanism
- Payment confirmation
- PostgreSQL job queue
- Background worker
- Concurrency testing
- Load testing and benchmarking
